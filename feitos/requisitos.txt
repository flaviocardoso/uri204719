-Requisitos Não-Funcionais:

o sisteme apresenta facilidade de Uso - por meio de uma interface simples, de fácil manuseio e assimilação.
o sistema proporciona rapizes ao usuário - através de acesso rápido de acesso e comunicação com a sistema.
o sistema garante a segurança para o usuário - devidas permissões de acesso para cada uma das janelas do sistema e determinando que tipo de ação os usuários teram acesso(como um consulta, inclusão, alteração ou exclusão dos dados no sistema);
o sistema é multiplataforma - o usuário pode ser acessado por meio de diversos dispositivos, dentre eles computadores, notebooks, tables, smart phones baseados windows, ios, android.
o sistema tem rastreabilidade pelo gps em seus carros - isso garante a localização precisa de seus carro e pelo controle.
o sistema apresenta a rastrabilidade de toda a operação - isso permite que seja registrada o nome do usuário, data, hora, dados inseridos, dados alterados e os dados excluídos.
o sistema em platama web e aplicação fica disponível 24 horas por dia.
o Sistema permite o sitema de mutipagamento - permitindo o pagamento via boleto, cartão, paypal, etc...
O sistema permite que o usuário escolha em pegar o carro alugado na locadora ou que seja entregue.
o Sistema mantém dados em servidores seguros.
o sistema mantém carros alugado com seguro.
O sistema permite a porcentagem de promoções envetuais entre 10% a 20%.
O sistema mantém um tempo estipulado de 1 a 2 horas para o teste drive.
O sistema só aceita agencias de controle de combustivel, de regudoras, de financeiras autorizadas.
O sistema permite a sua manuntenção e controle de relatórios de erros.
O sistema permite a manuntenção de documentos.
O sistema auto verificasse para a correção de erros.
O sistema que tem telas do controle da interface on-line devem ter informação de ajuda associado em todas as funções para que o usuário saiba exatamente o que está fazendo.
O sistema deve supostar pelo menos 1000 usuários on-line.
O sistema precisa de acesso a internet para as plataformas web e aplicativos.
O sistema permite um raio de 10km para locadora mais próxima para aluguar carro via internet.


-Requisitos inversos:
O sistema não permite que o usuário que usuário cadastre mais de uma vez.
O sistema não permite que o usuário apague seu cadastro que tem pendencias.
O sistema não permite o aluguer de um carro com usuário que esteja longe da área de cobertura do raio de atuação.
